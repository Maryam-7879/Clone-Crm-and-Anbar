# راهنمای بروزرسانی دیتابیس

## نصب اولیه
برای نصب اولیه دیتابیس، فایل `schema.sql` را در phpMyAdmin اجرا کنید.

## بروزرسانی‌های مهم

### بروزرسانی payment_method (ضروری)
اگر خطای `Data truncated for column 'payment_method'` دریافت می‌کنید، این دستور را اجرا کنید:

```sql
ALTER TABLE sales MODIFY COLUMN payment_method ENUM('cash', 'card', 'transfer', 'cheque', 'installment');
```

### نحوه اجرا
1. وارد phpMyAdmin شوید
2. دیتابیس `crm_system` را انتخاب کنید
3. روی تب "SQL" کلیک کنید
4. دستور بالا را کپی و paste کنید
5. روی "اجرا" کلیک کنید

### بررسی موفقیت
پس از اجرا، باید پیام موفقیت نمایش داده شود. حالا فرم فروش بدون خطا کار خواهد کرد.

## نکات مهم
- همیشه قبل از اجرای دستورات SQL، از دیتابیس backup بگیرید
- اگر خطایی دریافت کردید، با مدیر سیستم تماس بگیرید
- این بروزرسانی فقط یک بار نیاز است
